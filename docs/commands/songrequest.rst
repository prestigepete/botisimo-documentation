!sr
============

**Default Permission:** Everyone

Request a song (works with the `music player <https://botisimo.com/account/music>`_).

Usage:
    !sr ``<search>``

Arguments:
    * ``search`` **<required>** - The search term(s) or video url to use to search YouTube
